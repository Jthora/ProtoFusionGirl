# iOS Content Port Reference

This folder tracks the iOS content companion repository for ProtoFusionGirl.

Source repository:
- https://github.com/Jthora/protofusiongirl_ios/

Purpose:
- Keep a clear pointer from this project to the iOS port effort.
- Store migration notes, parity checklists, and export/import artifacts used between repositories.

Suggested workflow:
1. Clone the iOS repository in a sibling directory:
   - `git clone https://github.com/Jthora/protofusiongirl_ios.git`
2. Compare assets/content between repositories.
3. Place any cross-repo mapping notes in this folder.

Optional (if you want this linked as a git submodule later):
- `git submodule add https://github.com/Jthora/protofusiongirl_ios.git iOS_content_port/protofusiongirl_ios`

Status:
- Reference established.
