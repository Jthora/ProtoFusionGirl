# System: Astrology Engine

> Cosmic cycle event generation — the world's heartbeat.

## Full Vision

The astrology engine maps the in-game calendar to cosmic cycles that influence event generation, Jane's emotional state, faction moods, and ley line resonance. It is the seed from which the world's emergent behavior grows.

### Core Concept

Astrology IS correspondences. A date maps to element/modality combinations that weight event probabilities and NPC behavior. The engine doesn't simulate planetary positions — it uses a lookup table of cosmic correspondences.

### Calendar Structure

The in-game timeline spans 2032-2036 (4 in-game years). Time passes at an accelerated rate (TBD — likely 1 real minute = 1 in-game day for prototype).

### Cosmic Cycle

A 12-phase cycle repeating every ~30 in-game days:

| Phase | Element | Modality | Effect on Events | Effect on Jane |
|-------|---------|----------|------------------|----------------|
| 1 | Fire | Cardinal | Increased conflict events | Aggressive, proactive |
| 2 | Fire | Fixed | Sustained tension | Determined, stubborn |
| 3 | Fire | Mutable | Transition, surprise events | Impulsive, scattered |
| 4 | Earth | Cardinal | Resource events, construction | Focused, practical |
| 5 | Earth | Fixed | Stability, slow build | Steady, cautious |
| 6 | Earth | Mutable | Disruption of stability | Anxious, adaptable |
| 7 | Air | Cardinal | Social events, faction shifts | Curious, diplomatic |
| 8 | Air | Fixed | Communication breakthroughs | Intellectual, detached |
| 9 | Air | Mutable | Misinformation, confusion | Scattered, creative |
| 10 | Water | Cardinal | Emotional events, rifts | Empathetic, protective |
| 11 | Water | Fixed | Deep healing, intense bonds | Focused on relationships |
| 12 | Water | Mutable | Dissolution, endings, new beginnings | Introspective, spiritual |

### Event Weight Matrix

Each cosmic phase adjusts probabilities:
```
eventProbability = baseRate × cosmicModifier[currentPhase][eventType]
```

Example: Fire Cardinal phase → conflict events 1.5x, resource events 0.7x, social events 0.8x

### History Log

Every event generated by the astrology engine is recorded:
```
{
  gameDate: number,
  cosmicPhase: number,
  eventType: string,
  eventId: string,
  outcome: 'resolved' | 'unresolved' | 'escalated',
  playerInvolved: boolean
}
```

This log IS the Timestream that the ASI can browse.

## Prototype Slice

### P2: Simple Calendar
- In-game time counter (day number from game start)
- 4 phases (one per element): Fire, Earth, Air, Water
- Each phase = 7 in-game days
- Phase determines a single event weight modifier

### P3: Event Integration
- Astrology modifier feeds into instability event generation
- Jane's emotional state gets a modifier from current phase
- History log starts recording events

### P5: Full 12-Phase Cycle
- All 12 phases with modality
- Rich event weight matrix
- Timestream browsable via history log
