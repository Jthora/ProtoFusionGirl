# Beta Readiness — Planning Hub

> **Status**: In planning
> **Current build state**: Tech demo — architecture solid, core loop incomplete
> **Date assessed**: 2026-03-17

---

## The Question

The game has strong systems and a clear identity. What it doesn't have yet is a **complete loop** — something a player can experience start-to-finish and walk away having felt the ASI relationship.

Beta testing requires at minimum: enter → understand role → guide Jane → face stakes → succeed or fail → want to return.

## Milestone Summary

| Milestone | Effort | Testable outcome |
|-----------|--------|-----------------|
| **Internal alpha** | ~3 days | First complete ASI loop: guide Jane, face stakes, win/lose |
| **Closed beta** | ~2 weeks | Polished first session, 2–3 mission arcs, trust relationship, combat feel |
| **Public beta** | ~6 weeks | Sprite art, audio, full mission set, leaderboards |

The 3-day internal alpha is the immediate target. Everything in [alpha-sprint.md](alpha-sprint.md) is scoped to that.

---

## The Test That Matters

After a beta session, a player should be able to say:

> *"I watched Jane almost get ambushed. I redirected her. She survived. The node stabilized. I felt responsible."*

That sentence is the success criterion for the entire beta. Every task in this plan exists to make that sentence possible.
